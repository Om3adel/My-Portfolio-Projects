This project develops an orbit propagator simulation tool designed to visualize satellite motion and support mission planning, particularly for educational purposes and LEO satellite power subsystem design. The system uses Two-Line Element (TLE) data from NASA/NORAD to perform orbit determination, position/velocity prediction, eclipse calculations, and ground track visualization. The solution is implemented using Python for mathematical computations—including orbital mechanics equations, ODE solvers, and Keplerian orbit calculations—and LabVIEW for creating an interactive Graphical User Interface with login authentication, orbital element display, and real-time satellite tracking animation. The propagator accounts for various perturbation factors such as Earth's oblateness, atmospheric drag, and solar pressure to improve accuracy, balancing computational complexity against precision requirements. The final deliverable enables users to immediately observe how different orbital parameters affect satellite trajectory, eclipse duration, and ground coverage, facilitating better understanding of orbital dynamics through interactive simulation.
